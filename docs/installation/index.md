# Installation Guide

The Installation Guide provides instructions for setting up a Carbide on your site. It includes two parts:

* **Site Reference Architecture**: Provides guidelines for hardware and configuration for Carbide managed sites.
* **Site Setup**: Outlines how to setup site dependencies for a Kubernetes-based Carbide install.
* **Building Carbide Containers**: Provides instructions for building the Carbide containers on a host machine or VM.
